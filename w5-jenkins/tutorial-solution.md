# Jenkins Tutorial
## Maven Integration
- Plugin: Maven Integration, GitHub
- Manual Set-up: 
  1. **Tools - JDK - Add JDK**
     - Input Name=JDK 17
     - Input JAVA_HOME=/usr/lib/jvm/java-17-amazon-corretto.x86_64

  2. **Tools - Maven - Add Maven**
  - Input Name=Maven
  - Input MAVEN_HOME=/opt/maven

  3. **Tools - Git**
  - Input Name=Git
  - Input Path to Git executable =git
